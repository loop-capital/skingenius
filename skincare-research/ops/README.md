# Operations Directory

This directory contains operational assets for the SKINgenius pre-launch phase.

## Files

- `BETA-WAITLIST-LANDING-PAGE.md` - Landing page copy for beta signups
- `EMAIL-NURTURE-SEQUENCE.md` - First 5 emails in welcome/onboarding sequence
- `LEGAL-FRAMEWORK.md` - Legal framework outline (ToS, Privacy Policy, etc.)
- `SUMMARY.md` - This summary document
- `.gitkeep` - Git tracking placeholder

## Purpose

These documents provide the foundational copy, messaging, and legal groundwork needed to launch the beta waitlist and begin user acquisition for SKINgenius, an AI-powered skin health platform.

All content should be reviewed by relevant stakeholders (marketing, product, legal) before public deployment.
