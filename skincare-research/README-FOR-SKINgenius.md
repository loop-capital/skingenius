## Getting Started

### Week 1: Foundation
- Review PRD-v1.md and MVP-SCOPE.md with tech lead
- Set up development environment (per MVP-SCOPE)
- Load initial data from DATA/ directory
- Review and approve content tone from CONTENT/ articles

### Week 2-4: Sprint 1 - Core MVP
- User authentication & profiles
- Skin analysis flow (photo capture → results)
- Basic results dashboard (8 metrics)
- Onboarding flow

### Week 5-8: Sprint 2 - Engagement
- Routine builder (basic version)
- Progress tracking (photo timeline)
- Content discovery (first 5 articles)
- Settings & preferences

### Week 9-10: Polish & Beta Prep
- Bug fixing & performance optimization
- Beta onboarding flow (email sequences)
- Final QA & user testing preparation
