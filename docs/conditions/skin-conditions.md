# Skin Condition Profiles for SKINgenius

This document provides clinically accurate profiles of the top 15 skin conditions that SKINgenius will address. Each profile includes:
- Name (medical + common names)
- Prevalence
- Pathophysiology
- Visual indicators
- Standard treatment protocol
- Active ingredients that help (with evidence levels)
- Ingredients to avoid
- Lifestyle factors
- When to see a doctor (red flags)
- Related biomarkers (for Basys Health integration)

## Conditions Covered:
1. Acne (comedonal, inflammatory, cystic)
2. Rosacea (types 1-4)
3. Hyperpigmentation (PIH, melasma, sun damage)
4. Eczema/Atopic Dermatitis
5. Psoriasis
6. Seborrheic Dermatitis
7. Contact Dermatitis
8. Aging (photoaging vs chronological)
9. Keratosis Pilaris
10. Folliculitis
11. Perioral Dermatitis
12. Vitiligo
13. Stretch Marks
14. Dark Circles
15. Sensitive Skin
